## Vue d’ensemble

myTinyTodo permet de Gérer efficacement vos listes de tâches.

### Caractéristiques

- Plusieurs listes
- Notes de tâches
- Tags (et nuage de tags)
- Les dates d'échéance
- Priorité (-1, 0, +1, +2)
- Différents tris dont le tri par glisser-déposer
- Recherche
- Mot de passe de protection
