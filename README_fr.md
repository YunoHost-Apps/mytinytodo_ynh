## Vue d’ensemble

### Features

- Multiple lists
- Task notes
- Tags (and tag cloud)
- Due dates
- Priority (-1, 0, +1, +2)
- Different sortings including sort by drag-and-drop
- Search
- Password protection
- Smart syntax improves creation of tasks
      (usage: /priority/ Task /tags/)
- Print-friendly CSS
- tyle for mobiles devices
